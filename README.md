# Hopwise App UI

UI design code for the Hopwise application.

## Project Structure

```
hopwise_app_ui/
├── README.md
├── assets/
│   ├── icon-2d.png
│   ├── icon-3d-1.png (to icon-3d-4.png)
│   └── filters/
│       ├── filter-food.png
│       ├── filter-drinks.png
│       ├── filter-icecream.png
│       └── filter-cafe.png
├── brand/
│   ├── hopwise-brand-guidelines.html
│   └── hopwise-brand-assets.html
├── mockups/
│   └── app-screens/
│       ├── hopwise-app-home-v1.html      ← Home (Discovery)
│       ├── hopwise-search-results.html   ← Search Results
│       ├── hopwise-restaurant-detail.html ← Restaurant Detail
│       ├── hopwise-ride-comparison.html  ← Ride Comparison
│       ├── hopwise-profile.html          ← Profile
│       └── hopwise-saved.html            ← Saved
├── production-app/  ← Production-ready application code
└── exploration/  (optional - design iterations)
    ├── hopwise-home-options.html
    └── hopwise-restaurant-filters.html
```

## Directory Overview

- **assets/** - All visual assets including icons and filter images
  - **filters/** - Category filter icons (food, drinks, ice cream, cafe)
- **brand/** - Brand guidelines and assets documentation
- **mockups/** - Design mockups and prototypes
  - **app-screens/** - HTML prototype screens
- **production-app/** - Production-ready application code
- **exploration/** - Design iterations and experimental UI variations

## Getting Started

Add your design assets and HTML prototypes to the appropriate directories following the structure above.
